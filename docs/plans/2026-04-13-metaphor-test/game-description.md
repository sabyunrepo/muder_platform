# 머더 미스터리 『메타포』 — 게임 설명서

## 게임 목표

머더 미스터리 『메타포』는 플레이어가 직접 이야기속의 인물이 되어 사건의 진상을 파헤치고, 미션을 수행하며 각자의 목표를 달성하는 롤플레잉 추리 게임입니다.

게임 도중, 다른 플레이어들과 밀담을 통해 정보를 은밀하게 교환하거나 획득할 수 있습니다.

플레이어들은 [점수 계산]에서 본인이 맡은 캐릭터가 달성할 수 있는 최대 점수를 획득할 수 있도록 플레이합니다.

> 캐릭터 대사를 플레이어가 실감나게 연기하면, 더 즐겁습니다.

## 게임의 흐름

1. **게임 세팅** — '사건의 배경' 확인
2. **캐릭터 선택 및 숙지**
3. **오프닝** — '진행' 채널
4. **자기소개**
5. **【1차 조사】** 15분
6. **【1차 토의】** 20분
7. **시크릿카드 공개** — '진행' 채널
8. **【2차 조사】** 15분
9. **【2차 토의】** 20분
10. **투표** 5분
11. **액션**
12. **엔딩** — '엔딩' 채널
13. **점수 계산**

## 주의 사항

- 모든 플레이어는 거짓말이 가능합니다. (스토리와 무관한 거짓말 남발 주의)
- 주어진 시간을 전부 활용하여, 인물들의 이야기를 최대한 풀어내는 것을 추천합니다.
- 역할에 몰입할수록, 더 재미있게 플레이 할 수 있습니다.

## 카드 시스템

- **단서 카드**: [1차 단서]와 [2차 단서]로 구성
- **아이템 카드(★)**: 게임 중 언제든지 사용 가능. 모든 플레이어에게 선언 후 즉시 사용. 먼저 선언한 플레이어의 아이템 사용 종료까지 다른 플레이어 사용 불가.
- 모든 카드는 【투표】 단계 전까지 다른 플레이어에게 **공개, 양도, 교환** 가능.

## 조사

- 항상 '저스티스' 플레이어가 제일 먼저 시작. 그 외 순서는 협의.
- 자신의 차례에 원하는 단서 카드 1장 획득. 조사 시간 종료 전까지 **4번 반복**.
- 카드 뒷면에 자신의 이름이 적힌 카드는 가져올 수 없음.
- 특별한 장소 카드는 특정 카드를 획득해야 얻을 수 있음.
- 같은 장소는 2번 조사할 수 없음.

## 토의

- 전체 토의로 진행. 밀담 불가.
- 범인 추리 + 이야기를 최대한 풀어내는 것이 목표.

## 밀담

- 2명 또는 3명이 따로 대화하는 것.
- 카드나 정보를 은밀하게 주고받거나 교섭 가능.
- **조사 단계에서만** 진행.
- 1명 남은 경우: 혼자인 플레이어는 즉시 원하는 밀담에 참여 가능 (거부권 없음).

## 투표

- 제한 시간 5분.
- 범인이라고 생각하는 인물에 투표.
- 가장 많은 득표를 받은 플레이어는 구속됨.

## 액션

- 행동 순서 1번인 플레이어부터 자신에게 주어진 액션 수행.

## 엔딩

- '엔딩' 채널에서 사건의 전말과 엔딩 확인.

## 점수 계산

- 개인 미션 점수 + 엔딩 점수 + 카드 점수 합산.
