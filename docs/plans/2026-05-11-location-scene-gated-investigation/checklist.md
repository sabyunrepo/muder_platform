# Location Tab Rework Checklist

## 목표

에디터의 장소 탭을 장면 기반 출현/숨김, 하위 장소 기반 조사, 전역 소모형 단서 획득 규칙으로 정렬해 제작자가 바로 운영 가능한 상태로 만든다.

## 포함 범위

- `locations` 탭 에디터 UI 리빌드
- 장소 노출 정책(씬/부모/숨김)
- 장소별 조사 모드/슬롯/순서 규칙
- 장소-단서 매핑 정규화 및 중복 방지
- 접근 제약 처리
- 정규화 마이그레이션

## 제외 범위

- 게임 런타임 전체 재작성(현재 단계는 에디터 중심)
- 실시간 서버 리그레이드(필수 연동 포인트만 정의)
- 단서 효과 엔진(기존 로직 변경 없이 연동만 정의)

## 완료 조건

- 장소와 단서 관계가 UI에서 단일 출처로 관리됨
- 출현/숨김 씬 구간으로 노출이 결정됨
- 하위 장소만 조사 단위로 동작함
- 단서 중복 매핑이 막힘
- 동시 조사 시 요청순 선점 및 소모성 규칙이 보존됨
