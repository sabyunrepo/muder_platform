⚠️ 결정 디테일 4건

1. 질문 보기 자동 채움? — 캐릭터 entity 가 자동으로 Q1 보기 / 단서 entity 가 자동으로 Q2 보기. 변경 시 동기화. 수동 추가도 가능?
2. 질문 종류? — 단일 선택 (지금 mockup) / 다중 선택 / 자유 입력 (정답 키워드 매칭) / 순서 정렬 / 진위 판단 (O/X)
3. 답 평가 단위? — 개인별 (각 플레이어가 답 → 자기 결말 받음) vs 팀 합의 (다수결 → 모두 같은 결말) vs 혼합
4. 분기 조건의 풍부도? — 정답/오답 기반 (지금 mockup) vs 답 자체 매칭 ("Q1=이영희" 처럼 정답 개념 없음) — 후자가 더 유연
