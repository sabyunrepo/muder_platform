사이드바 하단 "🌐 글로벌 시스템" 클릭하면 보이는 페이지. 게임 1번 설정 = 모든 entity 에 적용.

⚖️ 결정 룰
"이 게임에서 누가 범인인지 정하는 방식" — 1번 결정, 모든 페이즈에 적용
🟢 voting (투표) ● ON
캐릭터들이 범인을 투표로 결정

무승부 시: 재투표

🟢 accusation (고소) ● ON

고소 횟수 제한: 게임당 1회
틀린 고소 페널티: 탈락
○ trade_clue (거래) ○ OFF

→ 끄면 설정 폼 안 보임 (entity 동적 섹션과 같은 패턴)
다른 글로벌 카테고리는 같은 형식:
  • 💬 커뮤니케이션: text_chat / group_chat / whisper / voice_chat / spatial_voice 5 모듈

  • 🎵 미디어: audio (1 모듈) — 게임 전체 BGM 기본값
