📋 베이스 🟢 모듈 동적 섹션 📜 markdown 컨텐츠

🟢 hidden_mission — 숨겨진 임무 (Markdown)

📋 베이스

🟢 room_exploration — 입장 룰
▾ 최대 동시 인원: 3 | 이동 쿨다운: 30초 | 다른 사람 위치 표시 ☑

▾ 선행 단서: [c2 편지] AND [c5 시체] → 자동 해제 | 알림 모드: 전원 공지

▾ 이 단서 + [c3 칼] = 새 단서 [c8 진실의 증거] 생성
🟢 clue_interaction — 사용·이전·중복 정책
▾ 다른 사람에게 양도 가능 ☑ | 중복 획득 시: 덮어쓰기 | 단서 레벨: 중급

▾ 라운드 시작 시 [c2 편지] → 김철수 / [c4 알리바이] → 전원
🟢 timed_clue — 주기 분배
▾ 5분마다 1개 / 풀: [c10] [c11] [c12] / 대상: 단서 적은 사람 / 최대 3회

▾ 일시정지 가능 ☑ / 강제 진행 ☑ / 시간 추가 ☑
